# c4s-api

Unofficial api client, types, and open-api schema for Clips4Sale adult video web store

## Installation

```bash
bun add c4s-api
```

## Usage

```typescript
import { greet } from 'c4s-api';

console.log(greet('World')); // Hello, World!
```
